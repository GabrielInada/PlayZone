O **PlayZone** é um **sistema web desenvolvido para o gerenciamento de campeonatos de futsal em ambiente acadêmico**, com foco em facilitar a organização de times, campeonatos, partidas e resultados dentro de instituições de ensino.

O projeto foi desenvolvido como parte de atividades acadêmicas, aplicando conceitos de **engenharia de software, desenvolvimento web e integração frontend/backend**.

link do youtube: https://youtu.be/Yme7WqIITl8

🌐 **Demo online:**  
https://play-zone-omega.vercel.app

---

## 🎯 Objetivo do Projeto

O objetivo do PlayZone é:

- Centralizar a gestão de campeonatos de futsal
- Facilitar o controle de times e partidas
- Automatizar a visualização de informações esportivas
- Aplicar na prática conceitos estudados em sala de aula

---

## 🧠 Funcionalidades

- 📋 Cadastro e visualização de campeonatos
- 👥 Gerenciamento de times
- 🏆 Organização de partidas
- 📊 Visualização de resultados
- 🧭 Navegação simples e intuitiva

## 🚀 Como Executar o Projeto Localmente

### Clone o repositório

```bash
git clone https://github.com/GabrielInada/PlayZone.git
cd PlayZone
```

### execute o Backend

```bash
cd backend
npm install
npm run dev
```

## Execute o Frontend

```bash
cd frontend
npm install
npm run dev
```
### Acesse no navegador:
 
```bash
- Frontend → http://localhost:3000
- Backend → http://localhost:3001
```
